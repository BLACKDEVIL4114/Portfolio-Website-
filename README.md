# Portfolio-Website-

This project is a premium, single-page portfolio website designed to showcase your skills as a Data Analyst & AI/ML Developer. Built using the modern React + Vite stack and styled with Tailwind CSS, it offers a high-performance, responsive experience with a sleek, dark-themed aesthetic.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
